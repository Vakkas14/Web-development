List of basic element:

| Tag              | Purpose                                  |
| ---------------- | ---------------------------------------- |
| `<html>`         | Root element of an HTML document         |
| `<head>`         | Contains metadata and page information   |
| `<title>`        | Sets the page title shown in browser tab |
| `<body>`         | Contains visible content                 |
| `<h1>` to `<h6>` | Headings                                 |
| `<p>`            | Paragraph                                |
| `<br>`           | Line break                               |
| `<hr>`           | Horizontal line                          |
| `<a>`            | Hyperlink                                |
| `<img>`          | Image                                    |
| `<ul>`           | Unordered list                           |
| `<ol>`           | Ordered list                             |
| `<li>`           | List item                                |
| `<div>`          | Block container                          |
| `<span>`         | Inline container                         |
| `<strong>`       | Important/Bold text                      |
| `<em>`           | Emphasized/Italic text                   |
| `<form>`         | Form container                           |
| `<input>`        | User input field                         |
| `<button>`       | Button                                   |
| `<label>`        | Label for form fields                    |
| `<table>`        | Table                                    |
| `<tr>`           | Table row                                |
| `<th>`           | Table heading                            |
| `<td>`           | Table data                               |

1. <html>
Purpose
The root element of every HTML document.
Example
<html>
</html>
Everything on the webpage is placed inside this tag.

2. <head>
Purpose
Stores metadata that is not displayed directly on the webpage.
Contains:
Title
CSS links
Meta tags
Scripts
Example
<head>
    <title>My Website</title>
</head>

3. <title>
Purpose
Displays the page title in the browser tab.
Example
<title>My Portfolio</title>
Result:
Browser Tab:
My Portfolio

4. <body>
Purpose
Contains all visible webpage content.
Example
<body>
    <h1>Hello World</h1>
</body>

5. Heading Tags
Tags
<h1></h1>
<h2></h2>
<h3></h3>
<h4></h4>
<h5></h5>
<h6></h6>
Purpose
Used for titles and subtitles.
Example
<h1>Main Title</h1>
<h2>Section Title</h2>
<h3>Subsection</h3>

6. <p>
Purpose
Creates a paragraph.
Example
<p>This is a paragraph.</p>

7. <br>
Purpose
Moves content to the next line.
Example
Hello<br>World
Output:
Hello
World

8. <hr>
Purpose
Creates a horizontal line.
Example
<hr>
Output:
-------------------

9. <a>
Purpose
Creates clickable links.
Syntax
<a href="URL">Text</a>
Example
<a href="https://google.com">Visit Google</a>

10. <img>
Purpose
Displays images.
Syntax
<img src="image.jpg" alt="Description">
Example
<img src="profile.jpg" alt="Profile Picture">

11. <ul>
Purpose
Creates an unordered (bulleted) list.
Example
<ul>
    <li>Apple</li>
    <li>Mango</li>
</ul>
Output:
Apple
Mango

12. <ol>
Purpose
Creates a numbered list.
Example
<ol>
    <li>Step 1</li>
    <li>Step 2</li>
</ol>
Output:
Step 1
Step 2

13. <li>
Purpose
Represents a list item.
Example
<li>HTML</li>

14. <div>
Purpose
Generic block-level container.
Example
<div>
    <h2>About Us</h2>
    <p>Company Information</p>
</div>
Commonly used for layouts.

15. <span>
Purpose
Generic inline container.
Example
<p>Hello <span>World</span></p>
Used for styling specific text.

16. <strong>
Purpose
Shows important text.
Example
<strong>Important Notice</strong>
Output:
Important Notice

17. <em>
Purpose
Emphasized text.
Example
<em>Highlighted Text</em>
Output:
Highlighted Text

18. <form>
Purpose
Creates a form.
Example
<form>
</form>

19. <input>
Purpose
Accepts user input.
Example
<input type="text">

20. <button>
Purpose
Creates clickable buttons.
Example
<button>Submit</button>

21. <label>
Purpose
Provides a label for form elements.
Example
<label>Name:</label>
<input type="text">

22. <table>
Purpose
Creates a table.
Example
<table>
</table>

23. <tr>
Purpose
Represents a table row.
Example
<tr>
</tr>

24. <th>
Purpose
Table heading cell.
Example
<th>Name</th>

25. <td>
Purpose
Table data cell.
Example
<td>John</td>
